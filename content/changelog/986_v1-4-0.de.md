## Hinzugefügt
- Zyklische Aufgaben können [konfiguriert](/de/server/periodic-tasks/configuring-tasks/) werden, um mit einem beliebigen Zeitplan vom Server ausgeführt zu werden.
- Eine neue zyklische Aufgabe erstellt eine [Sicherung der Datenbank](/de/server/periodic-tasks/disabled/database-backup).
- Der ["database-backup"](/de/client/manager/backup-management/) Befehl erlaubt das manuelle Erstellen und Wiederherstellen einer Sicherung.
- Die vordefinierte Nutzerrolle "robot" erleichtert das konfigurieren eines [Roboternutzers](/de/client/manager/access-management/robot-users/).
- Die Befehle ["csaf download"](/de/client/consumer/csaf-docs/#herunterladen), ["csaf get"](/de/client/consumer/csaf-docs/#anzeigen) und ["bom match"](/de/client/consumer/boms/#zuordnen) akzeptieren ein "--provider-metadata" Argument um die Interoperabilität mit CSAF Providern zu gewährleisten, die die Vorschläge des Standards bezüglich des URL Pfades ignorieren.

## Geändert
- Das Herunterladen von Dokumenten von einem nicht-BOMnipotent CSAF Server respektiert das aktuell gespeicherte Inventar.

## Behoben
- Ein neuer Schlüssel wird von ["user request"](/de/client/basics/account-creation/#schlüssel-an-beliebiger-stelle-speichern) korrekt an einem explizit angegebenen Pfad erzeugt, auch wenn ein anderer Schlüssel bereits unter dem Default-Pfad existiert.
